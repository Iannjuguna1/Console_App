# MyFirstApp Summary

## Overview
MyFirstApp is a simple C# console calculator. It prompts the user for two numbers and an operator, performs the calculation, and prints the result.

## What It Does
- Accepts two numeric inputs.
- Accepts an operator: `+`, `-`, `*`, or `/`.
- Computes and displays the result.
- Handles division by zero by returning `NaN`.
- Exits early on invalid operators.

## How To Run
From the project directory:
```powershell
dotnet run
```

## Project Structure
- `Program.cs` — Console app entry point and calculator logic.
- `MyFirstApp.csproj` — .NET project file.
- `MyFirstApp.sln` — Solution file.
